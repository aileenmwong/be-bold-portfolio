Explore

Technologies Used

About the Build

Explore is full-stack app that helps plan a roadtrip to all of the US National Parks. Using the MVC framework, I was able to pass data from the back end to the front end successfully. Users can search for activities and park names on the search page, which queries the National Parks Services API and the data renders in both a list view and through Google Maps.

Date a Dog

Technologies Used

HTML/CSS/JavaScript/jQuery

About the Build

Date A Dog is a javascript dating game to find your new best friend. The game features Scout the dog, who asks you questions as if you are on a first date. Based on answers, the users accumulate a certain amount of hearts. If enough hearts are earned, the user wins a date with Scout. The code was written in Vanilla Javascript and refactored in jQuery.
